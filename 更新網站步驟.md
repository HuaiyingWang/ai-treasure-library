# GitHub Pages 部署與更新指南

## 一、建立新 Repo 並首次部署

### 1. 在 GitHub 網頁建立 Repo

1. 前往 [github.com](https://github.com) 並登入
2. 右上角 **+** → **New repository**
3. Repository name 填入專案名稱（例如：`ai-treasure-library`）
4. 選 **Public**
5. **不要**勾選任何初始化選項（README、.gitignore 等）
6. 按 **Create repository**

### 2. 在 CMD 初始化並推送

```cmd
cd C:\Users\User\Desktop\ai_treasure_library
git init
git add index.html
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/你的帳號/ai-treasure-library.git
git push -u origin main
```

> 把「你的帳號」換成你的 GitHub 用戶名。推送時輸入帳號與 Personal Access Token（非 GitHub 密碼）。

### 3. 開啟 GitHub Pages

1. 進入 repo 頁面 → 上方 **Settings**
2. 左側 **Pages**
3. Branch 選 `main`（或 `master`），資料夾選 `/ (root)`
4. 按 **Save**

等約 1-2 分鐘，網站即上線。

---

## 二、更新網站步驟

每次修改 `index.html` 後，在 CMD 依序執行以下指令即可更新網站。

## 指令

```cmd
cd C:\Users\User\Desktop\ai_treasure_library
git add index.html
git commit -m "更新說明"
git push
```

> 把「更新說明」換成這次改了什麼，例如：`git commit -m "新增工具卡片"`
